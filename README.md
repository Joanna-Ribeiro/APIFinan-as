# APIFinanças
API desenvolvida para controle financeiro pessoal. RESTful API que permite:  
Cadastrar Usuário,
Fazer Login,
Detalhar Perfil do Usuário Logado,
Editar Perfil do Usuário Logado,
Listar categorias, 
Listar transações, 
Detalhar transação, 
Cadastrar transação,
Editar transação, 
Remover transação, 
Obter extrato de transações.
